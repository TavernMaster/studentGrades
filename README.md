# Запуск

### 1. npm i

### 2. настроить подключение к БД (postgres) в src/config/database.json (По умолчанию: development)

### 3. npx sequelize-cli db:migrate - для миграции DB

### 4. npm start

### 5. Сервер запустится на localhost:8080

# GET /log Параметры пагинации

### 1. page=1

### 2. limit=10
